This is a boilerplate project using the following tech stack: node.js, express, mongodb, mongoose, backbone.js, require.js, dust.js, and bootstrap. It has barebones ReSTful API logic already written.


Installation
============

1. Go to nodejs.org to install the appropriate distribution of node.js for your system.
2. In a command shell, navigate to the project root directory and enter the command, "node app.js" to start the server.
3. Open http://localhost:3000/ in a browser.
